---
title: "Get The Financial Relations Manager of an Organization"
api: "GET https://api.schoolablesandbox.com/graphql/core/user"
---
Returns the details of the Financial Relations Manager of an organization. This query is on the [Core User](https://api.schoolablesandbox.com/graphql/core/user) endpoint both on Sandbox and Production.

<Tip>It's important to check out the [Authentication](api-reference/authentication) and [Caveat](api-reference/authentication) page before proceeding to use this page.  </Tip>

### Arguments

The ```getFinancialRelationsManager``` query accepts the ```GetFinancialRelationsManagerInput``` argument which is an object that contains only one item; ```organizationId```.

<ResponseField name="GetFinancialRelationsManagerInput" type="GetFinancialRelationsManagerInput!">
  <Expandable title="properties">
    <ParamField query="organizationId" type="ID" required>
        This is the ID assigned to an organization.
    </ParamField>
  </Expandable>
</ResponseField>


### Returns

The ```getFinancialRelationsManager``` query returns the following parameters;

<ResponseField name="firstName" type="string" required>
    This is the registered first name of the Financial Relations Manager
</ResponseField>
<ResponseField name="middleName" type="string">
    This is the registered middle name of the Financial Relations Manager 
</ResponseField>
<ResponseField name="lastName" type="string">
    This is the registered last name of the Financial Relations Manager 
</ResponseField>
<ResponseField name="id" type="ID" required>
    This is the ID assigned to the Financial Relations Manager.
</ResponseField>
<ResponseField name="familyId" type="ID">
    This is the ID of the family to which the Financial Relations Manager belongs.
</ResponseField>
<ResponseField name="schoolableId" type="ID">
    This is the schoolable ID of the Financial Relations Manager belongs.
</ResponseField>
<ResponseField name="familyRole" type="string">
    This is the role of the Financial Relations Manager in the family to which it belongs.
</ResponseField>
<ResponseField name="phoneNumber" type="string">
    This is the phone number of the Financial Relations Manager.
</ResponseField>
<ResponseField name="alternatePhoneNumber" type="string">
    This is the backup phone number of the Financial Relations Manager.
</ResponseField>
<ResponseField name="phoneNumberVerified" type="boolean" required>
    This boolean checks if the phone number of Financial Relations Manager is verified.
</ResponseField>
<ResponseField name="birthDay" type="number" required>
    This is the birthday of the Financial Relations Manager.
</ResponseField>
<ResponseField name="birthMonth" type="number" required>
    This is the birthday of the Financial Relations Manager.
</ResponseField>
<ResponseField name="birthYear" type="number" required>
    This is the birthday of the Financial Relations Manager.
</ResponseField>
<ResponseField name="bvn" type="string">
    This is the bank verification number(BVN) of the Financial Relations Manager.
</ResponseField>
<ResponseField name="nin" type="string">
    This is the national identification number(NIN) of the Financial Relations Manager.
</ResponseField>
<ResponseField name="emailAddress" type="string">
    This is the email address of the Financial Relations Manager.
</ResponseField>
<ResponseField name="emailVerified" type="boolean" required>
    This boolean checks if the email address of Financial Relations Manager is verified.
</ResponseField>
<ResponseField name="gender" type="UserGender!" required>
    This is the gender of the Financial Relations Manager.
</ResponseField>
<ResponseField name="status" type="UserStatus!" required>
    This is the user status of the Financial Relations Manager. It could be either of three enums; ```ACTIVE```, ```INACTIVE``` or ```DELETED```.
</ResponseField>
<ResponseField name="referralCode" type="string">
    This is the referral code assigned to the Financial Relations Manager.
</ResponseField>
<ResponseField name="refereeCode" type="string">
    This is the referee code assigned to the Financial Relations Manager.
</ResponseField>
<ResponseField name="isTransactionPinSet" type="boolean" required>
    This boolean checks if the Financial Relations Manager has a transaction pin or not.
</ResponseField>
<ResponseField name="profileImageUrl" type="string">
    This is the link to the profile image of the Financial Relations Manager.
</ResponseField>
<ResponseField name="createdAt" type="float" required>
    This is date the family was created. It is a float equivalent to what [Date.prototype.getTime()](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Date/getTime) returns.
</ResponseField>
<ResponseField name="updatedAt" type="float">
    This is date the family was last updated. It is also a float equivalent to what [Date.prototype.getTime()](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Date/getTime) returns.
</ResponseField>
